Modern file manager for macOS

Folderium

A native macOS file manager with dual-pane layout, inspired by Marta but open-source. Built entirely with Swift and SwiftUI.

View on GitHub Download

Powerful Features

Everything you need for efficient file management on macOS

Dual-Pane Interface

Two-pane layout for efficient file management with resizable panes and active pane highlighting.

Advanced Search

Real-time filtering, per-pane search, and system-wide Spotlight integration with regex support.

File Operations

Copy, cut, paste, rename, delete with drag & drop support and conflict resolution.

Archive Support

Compress and extract from ZIP, RAR, 7Z, TAR, GZIP, BZIP2, XZ, CAB, ISO, and more.

Terminal Integration

Embedded terminal with two-way directory sync and full command history support.

Cloud Storage

OneDrive, Dropbox, Google Drive, iCloud integration with symbolic link resolution.
